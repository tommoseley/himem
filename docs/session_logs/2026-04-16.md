# Session Log — 2026-04-16

## Scope

First development session for Hi Mem (formerly Memory Stream). Evaluated binder, built iOS app from scratch, deployed to device.

## Decisions

- Took project out of The Combine pipeline — developing directly
- SwiftUI over UIKit
- Anthropic Claude API (Sonnet) for cloud entity extraction, topic inference, inference summaries
- Apple NaturalLanguage framework for offline local NER
- API key stored in iOS Keychain, not source code
- Core Data for local persistence (6 entities)
- Topics are user-controlled — AI suggests, user approves via dialog
- Entity tags normalized: digits for numbers, singular form, 1-4 words max
- Voice recordings saved on-device with share button for export; configurable in Settings
- Renamed from Memory Stream to Hi Mem (bundle ID: com.himem.app)

- StorageService — Core Data stack, CRUD operations
- KeychainService — secure API key storage

- ProcessingEngine — hybrid orchestrator (cloud when connected, local when not)
- SpeechService — iOS Speech Framework with simultaneous audio file recording
- AudioPlayerService — AVAudioPlayer playback + share sheet
- ConnectivityMonitor — NWPathMonitor
- TopicApprovalService — queues new topic suggestions for user approval
- SearchEngine — Core Data queries with entity-type filtering
